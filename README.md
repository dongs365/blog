### 拉取代码
```
git clone 
```
### 安装hexo
```
npm install -g hexo-cli
```
### 查看版本
```
hexo -v
```
### 安装插件
```
npm install X –save
```
###  插件更新
```
npm install -g npm-check-updates
ncr-u
```
###  生成静态页面
```
hexo g
```
###  清理静态页面
```
hexo clean
```
###  部署页面
```
hexo d
```
###  本地启动
```
hexo s
```
